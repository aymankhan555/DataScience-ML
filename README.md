# Mental Health in Tech Survey - ML Project

This project predicts mental health treatment from a tech survey dataset.

## What I did:
- Cleaned gender data and filled missing values
- Tried 4 ML models (Random Forest, XGBoost, Logistic Regression, KNN)
- Compared model accuracy and F1 scores
- Showed confusion matrices, ROC curves, and feature importance

## How to run:
1. Download dataset from Kaggle: [Kaggle Mental Health Survey Dataset](https://www.kaggle.com/datasets/osmi/mental-health-in-tech-survey)  
2. Run the Python script or notebook
