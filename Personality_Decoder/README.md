# 🧠 Personality Decoder

This project predicts personality traits using social behavior data from the Kaggle Playground Series S5E7 dataset.

### 🔍 Key Features
- Smart imputation of missing values using most frequent strategy
- One-hot encoding of categorical variables
- Comparison of ML models: XGBoost, Random Forest, CatBoost, SVM
- Final predictions and submission file for Kaggle

### 📂 Files
- `personality_decoder.ipynb`: Full notebook with all steps
- `submission.csv`: Final test predictions (optional)

### 📊 Model Performance
Best F1 Score (SVM): **0.9665**

---

📌 **Dataset**: [Kaggle Playground Series - S5E7](https://www.kaggle.com/competitions/playground-series-s5e7)
